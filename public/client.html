Cette page existe que pour la configuration d'exemple : config/config_example.json.

Le contenu de cette page ne s'est pas correctement affiché.

Pourquoi ?
Cette page contient un script menant à la destination du répertoire client/.
Le fichier de configuration qu'exécute actuellement le serveur ne contient pas la destination du répertoire client/ ; il faut donc la définir dans votre fichier de configuration : config/your_config_file.json.

Le fichier de configuration par défaut est située ici : config/config_default.json ;
La configuration qui permettrait de faire fonctionner cette page correctement est : 

{
  ...
  "shared": {
    "paths": {
      ...
      "static": "public/",
      "client": "client/"
    }
  },
  ...
}

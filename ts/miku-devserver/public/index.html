Crée le 25 Février 2021

La destination de ce répertoire est définie dans la configuration du fichier `./config/config_default.json` 

shared.paths.static

$ cat ./config/config_default.json

{
  "env": "development",
  "shared": {
    "paths": {
      "static": "public/"
    }
  }
}
